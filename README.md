# Python Scheduler Application

A lightweight Python application that automates the execution of scheduled **Excel VBA macros**, eliminating manual clicks and reducing the risk of missed deadlines.

Designed for recurring business processes such as financial reporting, data refreshes, and reconciliation tasks.

---

## Overview

This application serves as a centralized scheduler that triggers Excel VBA macros at predefined times or intervals.  
It is built to run silently in the background, generate execution logs, and support multiple workbooks in a single workflow.

The goal is to provide a reliable and low-maintenance automation solution for enterprise environments.

---

## Key Features

- ✅ Automated execution of scheduled Excel VBA macros  
- ✅ Timestamped logs for every run  
- ✅ Supports multiple macros across multiple workbooks  
- ✅ Silent execution (no UI interaction required)  
- ✅ IT-friendly packaging for enterprise deployment  
- ✅ Suitable for recurring financial and operational processes  

---

## Use Cases

- Monthly / weekly financial reports  
- Automated data refreshes  
- Reconciliation and validation tasks  
- Batch macro execution across departments  

---

## High-Level Architecture

```text
Scheduler (Python)
        ↓
Macro Configuration
        ↓
Excel Workbook(s)
        ↓
VBA Macro Execution
        ↓
Execution Logs (Timestamped)
```
